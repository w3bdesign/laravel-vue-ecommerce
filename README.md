# Laravel eCommerce

 Creating an eCommerce site with Laravel 8 and Vue 3.

 ## Features

 - Laravel 8
 - Vue 3
 - VueX state management
 - Composition API
 - Laravel Scout
 - Laravel tests with CircleCI integration
